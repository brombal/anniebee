If you are having any trouble printing these patterns, follow these instructions:

(For Netscape Users)

1. Select 'File' from the menu bar at the top
2. Select 'Page Setup' or whatever allows you to change the way your page looks when you print it.
3. Set all the margins to '0'
4. Make sure any other settings are off (the boxes next to them are empty)
5. Click Ok.
6. Click on the pattern that you want to print.
7. Select 'File' from the menu bar at the top
8. Select 'Print...'
9. Click Ok if any windows appear.

(For Internet Explorer Users)

1. Select 'File' from the menu bar at the top
2. Select 'Page Setup' or whatever allows you to change the way your page looks when you print it.
3. Set all the margins to '0' (they'll default to the smallest supported values)
4. You may also need to empty the Header and Footer settings as well
5. Click Ok.
6. Click on the pattern that you want to print.
7. Select 'File' from the menu bar at the top
8. Select 'Print...'
9. Click Ok if any windows appear.

If you need to resize your pattern, use the following formula:
(With gratitude to Hilary Berzins of England, who gave this to me, for I am one of the truly mathematically challenged of the world....)
1. Print out your pattern.
2. Measure the inch line in fractions of an inch.
3. Turn your fraction upside down.
4. Multiply by 100 to turn it into a percentage.

Example:
Your line measures 3/4 inch
Turn it upside down
(4/3 = 1.333333)
Multiply by 100
(133%)
(3/4 x 4/3 x 100 = 100% which is full size)
